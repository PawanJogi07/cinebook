# ✨ Features

✅ User Authentication (Login / Register)  
✅ Movie Search System  
✅ Trending Movies Section  
✅ Real-Time Seat Selection  
✅ Razorpay Payment Integration  
✅ Ticket Booking System  
✅ PDF Ticket Download  
✅ Mobile Responsive UI  
✅ Real-Time Booking Flow  
✅ Email Confirmation Support  
✅ Admin Panel  
✅ Dark Dark Mode UI
