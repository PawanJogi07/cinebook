# 🛠️ Tech Stack

## Frontend

* React.js
* Tailwind CSS
* React Router DOM
* Axios
* jsPDF
* Razorpay SDK

## Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Nodemailer
* Socket.io

## Deployment

* Vercel (Frontend)
* Render (Backend)
* MongoDB Atlas
